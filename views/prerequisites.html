Prerequisites

Before installing, there are certain tools, applications, and hardware you need to run a community server. It is important to follow these, unless you want a powerpoint slideshow as a server.

Hardware

All of this is based off what Oki recommends.

Hardware Recommended
Memory 4GB
Disk 2GB (I'm guessing this is if your not storing players stats)
OS Only Windows

32 vs 32 / 60 Hz

Hardware Minimum Recommended
CPU 2 Threads - 3.2Ghz Intel Pentium G3258 (found one for $5)
Network 10-20Mbit/s peak

64 vs 64 / 60 Hz

Hardware Minimum Recommended
CPU 2 Threads - 3.4Ghz/3.6Ghz Intel Pentium G4560
Network 40-50Mbit/s peak

127 vs 127 / 60 Hz

Hardware Minimum Recommended
CPU 4 Threads - 3.8Ghz Intel i3-4370
